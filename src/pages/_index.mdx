## API 服务 

```shell title="安装"
pnpm i @raopics/use
```

```js title="使用"
import EagleUse from "@raopics/use";

EagleUse({
  library: "xxx",
});
```

:::tip
`${host}/r` 可随机获取一张图片。
:::
